网站概念

服务器：也称伺服器，是提供计算服务的设备。与通用计算机类似，但是在处理能力、稳定性、可靠性、安全性、可扩展性、可管理性等方面要求较高。根据服务类型不同分为文件服务器、数据库服务器、应用程序服务器、WEB服务器等。

Web服务器：提供web服务（网站访问），需要安装web服务软件，如Apache、tomcat、iis等。

IP：Internet Protocol，网络互联协议。IP地址具有唯一性。

域名：Domain Name，由一串用点分隔的名字组成的Internet上某一台计算机或计算机组的名称，是一个IP地址的”面具“，便于记忆和沟通。

特殊IP：127.0.0.1，代表本机

特殊域名：localhost

DNS：Domain Name System，域名系统。因特网上作为域名与IP地址相互映射的分布式数据库，方便用户使用互联网时使用域名即可访问。

端口：Port，设备与外界通讯交流的出口。端口分为虚拟端口和物理端口，物理端口又称接口，是计算机上的插口；虚拟端口用于访问计算机里某个具体的程序。

静态网站访问流程：

  1. 在个人电脑里的浏览器输入域名，浏览器首先会尝试在本地DNS（hosts）找到域名相对应的IP，如果没有，再访问网络DNS尝试解析域名，获取域名对应的IP地址
  2. 拿到IP地址后， 浏览器再通过IP访问服务器
  3. 访问到服务器后，http请求根据端口找到Apache服务软件
  4. Apache根据URI相对路径找到对应的资源并返回给个人电脑

动态网站访问流程：

  1. 在个人电脑里的浏览器输入域名，浏览器首先会尝试在本地DNS（hosts）找到域名相对应的IP，如果没有，再访问网络DNS尝试解析域名，获取域名对应的IP地址
  2. 拿到IP地址后， 浏览器再通过IP访问服务器
  3. 访问到服务器后，http请求根据端口找到Apache服务软件
  4. Apache启用PHP引擎解析动态生成网页资源，这个过程中PHP可能会访问数据库服务器获取数据，生成动态网页后Apache将网页返回给个人电脑
